# byte-utils

[![npm version](https://img.shields.io/npm/v/byte-utils.svg)](https://www.npmjs.com/package/byte-utils)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Byte size formatting and parsing (KB, MB, GB)

## Installation

```bash
npm install byte-utils
```

## Functions

- `formatBytes` — Format bytes to human-readable string
- `parseBytes` — Parse human-readable byte string to bytes
